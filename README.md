# Fundamentos de Ingeniería de Software

## Metas

- Comprender cómo se construye el software, no solo cómo escribir código.
- Escriba programas limpios y legibles utilizando variables, flujo de control y programación orientada a objetos
- Comprender la arquitectura básica del software y los principios de diseño.
- Aplicar prácticas de ingeniería: pruebas, control de versiones y documentación.

## Estructura

### 1. ¿Qué Es La Ingeniería De Software?

**Meta:** Construye el modelo mental.

#### Temas

- Qué es la ingeniería de software (frente a la programación).
- Cómo se construye el software del mundo real.
- Ciclo de vida del software (idea → código → implementación → mantenimiento).
- Tipos de software (web, móvil, integrado, backend).
